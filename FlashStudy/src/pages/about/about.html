Project 3 (Creating the layout)

(In this about, I'm reffering to each page by it's title not it's file name).

on the set settings pop over page, clicking "add cards" will bring you to the "new cards" page.

Project 4 (Authentication and Database)

The implemented functionality for this project:
1. Authentication - signup, login, change password, reset password, and logout.
2. add sets and cards for each set
3. delete sets and cards
4. personalize set - change the set name and icon
5. ability to hide/show the back text on cards, and reset all of the cards to hidden
6. cards and sets only show for the account that they were created on
